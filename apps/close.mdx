- [Auth Documentation](https://developer.close.com/topics/authentication-oauth2/)

## ⚙️ Configuration

<Tip>API Token can be obtained from the [Close Developer Dashboard](https://app.close.com/settings/integrations/api_keys/).</Tip>


```
https://backend.composio.dev/api/v1/auth-apps/add
```

Generate an API token and copy it. Go to Composio's Integrations page, find Close, and click "+ new integration". Now click on the "+ Add Account" button, Add your API token and click "Grant Access" to connect your account.
