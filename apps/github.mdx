## ‚öôÔ∏è¬†Configuration

<Tip>
New App can be created at [GitHub Developer Settings](https://github.com/settings/developers).
</Tip>

Once you create the App, find the _Authorization callback URL_ field and configure the below Redirect URL:

```
https://backend.composio.dev/api/v1/auth-apps/github/connection/oauth2/add
```

After registering the app, copy the Client ID from the app page and generate/copy the Client Secret. On Composio's [supported apps](https://app.composio.dev/apps) page, find GitHub and click "Enable". Paste your Client ID and Secret, then Save to connect GitHub.
