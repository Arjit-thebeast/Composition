> Jira uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.atlassian.com/cloud/jira/platform/oauth-2-3lo-apps/)
- [API Documentation](https://developer.atlassian.com/cloud/jira/platform/rest/v3/intro/#about)

## ⚙️ Configuration

<Tip>New App can be created on [Jira Developer Console](https://developer.atlassian.com/).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and the Client Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Jira and add a new integration. Paste your Client ID and Secret, then Save to connect Jira.


## ⚒️ Actions

Jira actions are not supported yet. In case, you’re looking for Jira actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Jira triggers are not supported yet. In case, you’re looking for Jira Triggers, please write to us at tech@composio.dev.
