> Miro uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developers.miro.com/docs/getting-started-with-oauth)
- [API Documentation](https://developers.miro.com/reference/api-reference)

## ⚙️ Configuration

<Tip>New App can be created on [Miro Developer Console](https://developers.miro.com/page/developer-hub#your-apps).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and the Client Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Miro and add a new integration. Paste your Client ID and Secret, then Save to connect Miro.
