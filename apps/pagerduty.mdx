> Pagerduty uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.pagerduty.com/docs/dd91fbd09a1a1-register-an-app)
- [API Documentation](https://developer.pagerduty.com/api-reference/)

## ⚙️ Configuration

<Tip>New OAuth App can be created on the [Pager Duty Developer Account](https://developer.pagerduty.com/sign-up/) -> Intergrations -> App Registration -> My Apps -> New App.</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and the Client Secre from the Client section in the App page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Pagerduty and add a new integration. Paste your Client ID and Secret, then Save to connect Pagerduty.


## ⚒️ Actions

Pagerduty actions are not supported yet. In case, you’re looking for Asana actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Pagerduty triggers are not supported yet. In case, you’re looking for Asana Triggers, please write to us at tech@composio.dev.
