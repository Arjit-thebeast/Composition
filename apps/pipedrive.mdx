> Pipedrive uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://pipedrive.readme.io/docs/marketplace-oauth-api)
- [API Documentation](https://developers.pipedrive.com/docs/api/v1)

## ⚙️ Configuration

<Tip>New App can be created on the [Pipedrive Developer Hub](https://karan-sandbox2.pipedrive.com/developer-hub).</Tip>

Configure the below URL in the _Callback URL_ field:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and the Client Secret from the OAuth & access scopes section in the App page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Pipedrive and add a new integration. Paste your Client ID and Secret, then Save to connect Pipedrive.
