> Sage uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.sage.com/accounting/guides/authenticating/authentication/)
- [API Documentation](https://developer.sage.com/api/accounting/api/)

## ⚙️ Configuration

<Tip>New App can be created on the [Sage Developer Portal](https://developerselfservice.sageone.com/user).</Tip>

Once you create the App, please configure the below Redirect URL in the _Callback URLs_ field:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Sage and add a new integration. Paste your Client ID and Secret, then Save to connect Sage.


## ⚒️ Actions

Sage actions are not supported yet. In case, you’re looking for Asana actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Sage triggers are not supported yet. In case, you’re looking for Asana Triggers, please write to us at tech@composio.dev.
