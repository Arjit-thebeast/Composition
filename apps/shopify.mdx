> Shopify uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://shopify.dev/docs/apps/auth/get-access-tokens/authorization-code-grant/getting-started)
- [API Documentation](https://shopify.dev/docs/api)

## ⚙️ Configuration

<Tip>New OAuth App can be created in the Apps section in the [Shopify Partner Dashboard](https://partners.shopify.com/organizations?).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from API Access section. On Composio's [Integrations](https://app.composio.dev/apps) page, find Shopify and add a new integration. Paste your Client ID and Secret, then Save to connect Shopify.


## ⚒️ Actions

Shopify actions are not supported yet. In case, you’re looking for Asana actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Shopify triggers are not supported yet. In case, you’re looking for Asana Triggers, please write to us at tech@composio.dev.
