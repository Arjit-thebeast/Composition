> Taskade uses OAUTH2 for authentication.
## 📔 Documentation

- [API Documentation](https://www.taskade.com/developers/api)

## ⚙️ Configuration

<Tip>New OAuth App can created on the [Taskade Settings page](https://www.taskade.com/settings/password).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the App Page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Taskade and add a new integration. Paste your Client ID and Secret, then save to connect Taskade.


## ⚒️ Actions

Taskade actions are not supported yet. In case, you’re looking for Asana actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Taskade triggers are not supported yet. In case, you’re looking for Asana Triggers, please write to us at tech@composio.dev.
