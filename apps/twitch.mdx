> Twitch uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://dev.twitch.tv/docs/authentication)
- [API Documentation](https://dev.twitch.tv/docs/api/)

## ⚙️ Configuration

<Tip>New App can be created on the [Twitch Developer Console](https://dev.twitch.tv/console/apps).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After creating the app, copy the Client ID and Client Secret from the App Page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Twitch and add a new integration. Paste your Client ID and Secret, then save to connect Twitch.


## ⚒️ Actions

Twitch actions are not supported yet. In case, you’re looking for Twitch actions, please write to us at tech@composio.dev.
