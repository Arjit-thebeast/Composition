> Wakatime uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://wakatime.com/developers#oauth2)
- [API Documentation](https://wakatime.com/developers)

## ⚙️ Configuration

<Tip>New App can be created on the [Wakatime Apps Page](https://wakatime.com/apps).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After creating the app, copy the App ID and App Secret from the App Page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Wave and add a new integration. Paste your App ID and Secret, then save to connect Wave.


## ⚒️ Actions

Wakatime actions are not supported yet. In case, you’re looking for Asana actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Wakatime triggers are not supported yet. In case, you’re looking for Asana Triggers, please write to us at tech@composio.dev.
