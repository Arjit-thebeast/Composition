> Wave Accounting uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.waveapps.com/hc/en-us/articles/360019968212-OAuth-2-0)
- [API Documentation](https://developer.waveapps.com/hc/en-us/categories/360001114072-API-Documentation)

## ⚙️ Configuration

<Tip>New OAuth App can be created on the [Wave Developer Portal](https://developer.waveapps.com/hc/en-us/articles/360019762711-Manage-Applications).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After creating the app, copy the Client ID and Client Secret from the App Page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Wave and add a new integration. Paste your Client ID and Secret, then save to connect Wave.
