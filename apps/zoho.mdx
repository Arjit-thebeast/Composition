> Zoho uses OAUTH2 for authentication.
## 📔 Documentation

- [Zoho OAuth Documentation](https://www.zoho.com/accounts/protocol/oauth/web-server-applications.html)
- [Zoho API Documentation](https://www.zoho.com/developer/rest-api.html)

## ⚙️ Configuration

<Tip>New OAuth App can be created on the [Zoho Developer Console](https://api-console.zoho.in/).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Secret from Client Secret section of the App page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Zoho CRM and add a new integration. Paste your Client ID and Secret, then save to connect Zoho CRM.
