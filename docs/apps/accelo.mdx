> Accelo uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://api.accelo.com/docs/?http#authentication)
- [API Documentation](https://api.accelo.com/docs/?http)

## ⚙️ Configuration

<Tip>New App can be created on the your Accelo account -> Configuration -> API -> Register Application.</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Accelo and add a new integration. Paste your Client ID and Secret, then Save to connect Accelo.


## ⚒️ Actions

Accelo actions are not supported yet. In case, you’re looking for Accelo actions, please write to us at tech@composio.dev.
