> Adobe uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.adobe.com/developer-console/docs/guides/authentication/UserAuthentication/)
- [API Documentation](https://developer.adobe.com/photoshop/photoshop-api-docs/)

## ⚙️ Configuration

<Tip>API Key can be obtained from the [Adobe Developer Console](https://developer.adobe.com/console).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Adobe and click"+ new integration" button. Paste your Client ID and Secret, then Save to connect Adobe.
