> Atlassian uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.atlassian.com/cloud/jira/platform/oauth-2-authorization-code-grants-3lo-for-apps/)
- [API Documentation](https://developer.atlassian.com/cloud/jira/platform/rest/v3/intro/)

## ⚙️ Configuration

<Tip>New App can be created on [Atlassian Developer Console](https://developer.atlassian.com/console/myapps/).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID from the app page and generate/copy the Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Atlassian and add a new integration. Paste your Client ID and Secret, then Save to connect Atlassian.


## ⚒️ Actions

Atlassian actions are not supported yet. In case, you’re looking for Atlassian actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Atlassian triggers are not supported yet. In case, you’re looking for Atlassian Triggers, please write to us at tech@composio.dev.
