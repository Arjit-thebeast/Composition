> Battle.net uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://develop.battle.net/documentation/guides/using-oauth)
- [API Documentation](https://develop.battle.net/documentation)

## ⚙️ Configuration

<Tip>New App can be created on the [Battle Net Developer Portal](https://develop.battle.net/access/clients).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Battle.Net and add a new integration. Paste your Client ID and Secret, then Save to connect Battle.Net.

## ⚒️ Actions
