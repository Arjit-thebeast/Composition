> Blackbaud uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.blackbaud.com/skyapi/docs/authorization)
- [API Documentation](https://developer.blackbaud.com/skyapi/products)

## ⚙️ Configuration

<Tip>New OAuth App can be created on the [Blackbaud Developer Apps page](https://developer.blackbaud.com/apps/).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Application ID and Application Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Blackbaud and add a new integration. Paste your Client ID and Secret, then Save to connect Blackbaud.
