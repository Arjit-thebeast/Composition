> Box uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.box.com/guides/authentication/oauth2/)
- [API Documentation](https://developer.box.com/reference/)

## ⚙️ Configuration

<Tip>New App can be created on [Box Developer Console](https://app.box.com/developers/console)</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID from the app page and generate/copy the Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Box and add a new integration. Paste your Client ID and Secret, then Save to connect Box.


## ⚒️ Actions

Box actions are not supported yet. In case, you’re looking for Box actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Box triggers are not supported yet. In case, you’re looking for Box Triggers, please write to us at tech@composio.dev.
