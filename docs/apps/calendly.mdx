> Calendly uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.calendly.com/api-docs/3cefb59b832eb-calendly-o-auth-2-0)
- [API Documentation](https://developer.calendly.com/api-docs)

## ⚙️ Configuration

<Tip>New App can be created on [Calendly Developer Console](https://developer.calendly.com/console/apps).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID from the app page and generate/copy the Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Calendly and add a new integration. Paste your Client ID and Secret, then Save to connect Calendly.


## ⚒️ Actions

Calendly actions are not supported yet. In case, you’re looking for Calendly actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Calendly triggers are not supported yet. In case, you’re looking for Calendly Triggers, please write to us at tech@composio.dev.
