<Tip>New App can be created on the [Epic Account Services page](https://dev.epicgames.com/portal/en-US/composio/products/composio/epic-account-services).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, Create the Client Secret in the [Product Settings page](https://dev.epicgames.com/portal/en-US/composio/products/composio/settings/clients) and copy it. On Composio's [Integrations](https://app.composio.dev/apps) page, find Epic Games and add a new integration. Paste your App Name and Secret, then Save to connect Epic Games.
