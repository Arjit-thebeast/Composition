> Figma uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://www.figma.com/developers/api#oauth2)
- [API Documentation](https://www.figma.com/developers/api)

## ⚙️ Configuration

<Tip>New OAuth App can be created on [Figma Developer Apps page](https://www.figma.com/developers/apps).</Tip>

Once you create the App, please configure the below Redirect URL in the _Callbacks_ field:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After creating the app, copy the Client ID and Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Figma and add a new integration. Paste your Client ID and Secret, then Save to connect Figma.


## ⚒️ Actions
