> Freshbooks uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://www.freshbooks.com/api/authentication)
- [API Documentation](https://www.freshbooks.com/api/start)

## ⚙️ Configuration

<Tip>New OAuth App can be created on [Freshbooks Developer Apps page](https://www.freshbooks.com/developers).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Freshbooks and add a new integration. Paste your Client ID and Secret, then Save to connect Freshbooks.


## ⚒️ Actions

Freshbooks actions are not supported yet. In case, you’re looking for Freshbooks actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Freshbooks triggers are not supported yet. In case, you’re looking for Freshbooks Triggers, please write to us at tech@composio.dev.
