> Freshdesk uses API_KEY for authentication.
## 📔 Documentation

- [Auth Documentation](https://support.freshdesk.com/en/support/solutions/articles/215517-how-to-find-your-api-key)
- [API Documentation](https://developers.freshdesk.com/api/)

## ⚙️ Configuration

<Tip>API Key can be obtained from the FreshDesk Profile Settings page -> View API Key.</Tip>


```
https://backend.composio.dev/api/v1/auth-apps/add
```

Copy the API Key and go to Composio's Integrations page, find FreshDesk, and click "+ new integration". Now click on the "+ Add Account" button, Add your API token and click "Grant Access" to connect your account.
