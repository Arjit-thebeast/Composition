> Front uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://dev.frontapp.com/docs/authentication)
- [API Documentation](https://dev.frontapp.com/reference/introduction)

## ⚙️ Configuration

<Tip>API Token can be obtained from the [Front API Dashboard](https://app.frontapp.com/settings/tools/api)</Tip>


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Front and add a new integration. Paste your Client ID and Secret, then Save to connect Front.


## ⚒️ Actions
