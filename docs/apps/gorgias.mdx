> Gorgias uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developers.gorgias.com/docs/set-up-oauth2-app-store)
- [API Documentation](https://developers.gorgias.com/reference/introduction)

## ⚙️ Configuration

<Tip>New App can be created on [Gorgias Developer Portal](https://partners.gorgias.com/login)</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID from the app page and generate/copy the Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Gorgias and add a new integration. Paste your Client ID and Secret, then Save to connect Gorgias.


## ⚒️ Actions

Gorgias actions are not supported yet. In case, you’re looking for Gorgias actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Gorgias triggers are not supported yet. In case, you’re looking for Gorgias Triggers, please write to us at tech@composio.dev.
