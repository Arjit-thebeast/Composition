> Gumroad uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://help.gumroad.com/article/280-create-application-api)
- [API Documentation](https://app.gumroad.com/api)

## ⚙️ Configuration

<Tip>New OAuth App can be created on [Gumroad Apps page](https://app.gumroad.com/settings/authorized_applications).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and the Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Gumroad and add a new integration. Paste your Client ID and Secret, then Save to connect Gumroad.


## ⚒️ Actions

Gumroad actions are not supported yet. In case, you’re looking for Gumroad actions, please write to us at tech@composio.dev.

## 🪝 Triggers

Gumroad triggers are not supported yet. In case, you’re looking for Gumroad Triggers, please write to us at tech@composio.dev.
