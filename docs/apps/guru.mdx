> Guru uses API_KEY for authentication.
## 📔 Documentation

- [Auth Documentation](https://developer.getguru.com/docs/getting-started)
- [API Documentation](https://docs.getguru.fitness/#api)

## ⚙️ Configuration

<Tip>API Token can be obtained from the [Guru Console](https://console.getguru.ai/)</Tip>


```
https://backend.composio.dev/api/v1/auth-apps/add
```

Generate an API token and copy it. Go to Composio's Integrations page, find Guru, and click "+ new integration". Now click on the "+ Add Account" button, Add your API token and click "Grant Access" to connect your account.
