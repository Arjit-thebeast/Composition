## ⚙️ Configuration

<Tip>API Key can be obtained from the [Hackerrank API Settings](https://hackerrank.com/x/settings/api)</Tip>


```
https://backend.composio.dev/api/v1/auth-apps/add
```

Generate an API Key and copy it. Go to Composio's Integrations page, find HackerRank, and click "+ new integration". Now click on the "+ Add Account" button, Add your API token and click "Grant Access" to connect your account.
