> Harvest uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://help.getharvest.com/api-v2/authentication-api/authentication/authentication/)
- [API Documentation](https://help.getharvest.com/api-v2/)

## ⚙️ Configuration

<Tip>New OAuth App can be created on the [Harvest Developers section](https://id.getharvest.com/developers).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the App page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Harvest and add a new integration. Paste your Client ID and Secret, then Save to connect Harvest.
