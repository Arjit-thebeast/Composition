> Klaviyo uses API_KEY for authentication.
## 📔 Documentation

- [Auth Documentation](https://developers.klaviyo.com/en/docs/set_up_oauth)
- [API Documentation](https://developers.klaviyo.com/en/reference/api-overview)

## ⚙️ Configuration

<Tip>New OAuth App can be created on [Klaviyo OAuth Apps page](https://www.klaviyo.com/oauth/client).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret. Go to Composio's Integrations page, find Klaviyo, and click "+ new integration". Now click on the "+ Add Account" button, Add your Client ID and Client Secret and click "Grant Access" to connect your account.
