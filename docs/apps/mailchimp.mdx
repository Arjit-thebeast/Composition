> Mailchimp uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://mailchimp.com/developer/marketing/guides/access-user-data-oauth-2/)
- [API Documentation](https://mailchimp.com/developer/marketing/api/)

## ⚙️ Configuration

<Tip>New App can be created on [Mailchimp Developer Console](https://us1.admin.mailchimp.com/account/oauth2/).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and the Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Mailchimp and add a new integration. Paste your Client ID and Secret, then Save to connect Mailchimp.


## ⚒️ Actions

Mailchimp triggers are not supported yet. In case, you’re looking for Mailchimp Triggers, please write to us at tech@composio.dev.
