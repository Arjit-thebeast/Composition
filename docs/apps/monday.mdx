> Monday uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.monday.com/apps/docs/oauth)
- [API Documentation](https://developer.monday.com/apps/docs/mondayapi)

## ⚙️ Configuration

<Tip>New App can be created on the [Monday Developer Center](https://composio-squad.monday.com/apps/manage?placement=dev_center).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and the Client Secret from app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Monday and add a new integration. Paste your Client ID and Secret, then Save to connect Monday.
