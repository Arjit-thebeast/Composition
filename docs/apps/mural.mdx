> Mural uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developers.mural.co/public/docs/oauth)
- [API Documentation](https://developers.mural.co/public/docs/)

## ⚙️ Configuration

<Tip>New App can be created on the [Mural Apps Page](https://app.mural.co/me/apps).</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID from the app page and generate/copy the Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Mural and add a new integration. Paste your Client ID and Secret, then Save to connect Mural.
