> Okta uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.okta.com/docs/guides/implement-oauth-for-okta/overview/)
- [API Documentation](https://developer.okta.com/docs/reference/)

## ⚙️ Configuration

<Tip>New App can be created on the Okta Admin Comsole -> Applications -> Create App Integration.</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID and Client Secret from the _Client Credentials_ section of the app page. On Composio's [Integrations](https://app.composio.dev/apps) page, find Okta and add a new integration. Paste your Client ID and Secret, then Save to connect Okta.


## ⚒️ Actions
