> Servicem8 uses OAUTH2 for authentication.
## 📔 Documentation

- [OAuth Documentation](https://developer.servicem8.com/docs/authentication)
- [API Documentation](https://developer.servicem8.com/docs/)

## ⚙️ Configuration

<Tip>New App can be created on the [ServiceM8 Dashboard](https://go.servicem8.com/) -> Developer page -> Add Item.</Tip>

Once you create the App, please configure the below Redirect URL in the _Callback URL_ section:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the App ID and App Secret from . On Composio's [Integrations](https://app.composio.dev/apps) page, find Servicem8 and add a new integration. Paste your App ID and Secret, then Save to connect Servicem8.
