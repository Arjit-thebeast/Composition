> Shortcut uses API_KEY for authentication.
## 📔 Documentation

- [API Documentation](https://developer.shortcut.com/api/rest/v3)

## ⚙️ Configuration

<Tip>API Key can be obtained on the [Shortcut API Token page](https://app.shortcut.com/settings/account/api-tokens).</Tip>


```
https://backend.composio.dev/api/v1/auth-apps/add
```

Once you generated the API token and copy it. Go to Composio's Integrations page, find Shortcut, and click "+ new integration". Now click on the "+ Add Account" button, Add your API token and click "Grant Access" to connect your account.
