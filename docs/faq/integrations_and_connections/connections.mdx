### Listing All Connections
To view all the connections associated with your Composio account, use the following command:
<Tabs>
<Tab title="Python">
<CodeGroup>
```python List all connections via Python
from composio import ComposioToolSet

composio_client = ComposioToolSet()

entity = composio_client.get_entity(id="YOUR USER'S ENTITY ID")

try:
    connected_account  = entity.get_connection(app="slack")
except Exception as e:
    print("No connected account found")#This code attempts to retrieve the connected account for Slack. 
```
</CodeGroup>
</Tab>
<Tab title="CLI">
<CodeGroup>
```bash List all connections via CLI
composio connections
```
</CodeGroup>
</Tab>
<Tab title="JS">
```javascript javascript
import { Composio } from "composio-core";

const client = new Composio(
  process.env.COMPOSIO_API_KEY
);

const connectedAccounts = await client.connectedAccounts.list();

console.log(connectedAccounts)
```

### Getting Connection Details
To get detailed information about a specific connection, use the get command followed by the connection ID. For example, to get details of a connection with the ID 123, you would use:

<Tabs>
<Tab title="Python">
<CodeGroup>
```python Get connection details via Python
from composio import ComposioToolSet

composio_client = ComposioToolSet()

print(composio_client.client.connected_accounts.get('123'))
```
</CodeGroup>
</Tab>
<Tab title="CLI">

<CodeGroup>
```bash Get connection details via CLI
composio connections get 123
```
</CodeGroup>
</Tab>
<Tab title="JS">
```javascript javascript
import { Composio } from "composio-core";

const client = new Composio(
  process.env.COMPOSIO_API_KEY
);

const connectedAccount = await connectedAccounts.get({ connectedAccountId: `123` });

console.log(connectedAccount)
```
