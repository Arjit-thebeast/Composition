### Listing All Integrations
To retrieve a complete list of your integrations, you can use the following code snippets.

<Tabs>
<Tab title="Python">
<CodeGroup>
```python List all integrations via Python
from composio import Composio

composio_client = Composio()

print(composio_client.integrations.get())
```
</CodeGroup>
</Tab>
<Tab title="CLI">
<CodeGroup>
```bash List all integrations via CLI
composio integrations
```
</CodeGroup>
</Tab>
<Tab title="JS">
```javascript javascript
import { Composio } from "composio-core";

const client = new Composio(
  process.env.COMPOSIO_API_KEY
);

const integrations = await client.integrations.list();

console.log(integrations)
```
</Tab>

### Getting Integration Details
To get detailed information about a specific integration, use the get command followed by the integration ID. For example, to get details of a connection with the ID 123, you would use:

<Tabs>
<Tab title="Python">
<CodeGroup>
```python Get integration details via Python
from composio import ComposioToolSet
toolset = ComposioToolSet()

print(toolset.client.integrations.get())
print(toolset.client.integrations.get_by_id(integration_id="ADD THE ID HERE")) #filter based on integration id
```
</CodeGroup>
</Tab>
<Tab title="CLI">

<CodeGroup>
```bash bash
composio integrations get 123
```
</CodeGroup>
</Tab>
<Tab title="JS">
```javascript javascript    
import { OpenAIToolSet } from "composio-core";

const openai_client = new OpenAIToolSet({
    apiKey: process.env.OPENAIKEY
});

console.log(openai_client.client.integrations.list()) //lists all integrations
console.log(await openai_client.client.integrations.get({integrationId:"ADD THE ID HERE"})) //filter based on integration id
```
