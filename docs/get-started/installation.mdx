<Tabs>
<Tab title="Python">
<Steps>
<Step title="Install Composio Package">
Install the official [Composio package](https://pypi.org/project/composio-core), this will also install the Composio CLI
```python
pip install composio-core
```
</Step>
<Step title="Login via CLI">
To use Composio tools, you need to use the API key, by exporting `COMPOSIO_API_KEY=<value>` in your terminal, storing it in .env, or just login via CLI
```bash 
composio login
```
After executing this, you will automatically be redirected to Composio dashboard, from where you can get the authentication code
</Step>
</Steps>
