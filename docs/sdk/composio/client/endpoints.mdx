# `class Endpoint`
<Card>
HTTP Endpoint object.

    Example:
        >>> v1 = Endpoint("v1")

        /v1
        >>> v1 / "api" / "users"
        /v1/api/users
        >>> v1.api.users
        /v1/api/users
        >>> v1.api.users(queries={"user": "John Doe"})
        /v1/api/users?user=John+Doe
