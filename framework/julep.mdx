**Composio and Julep together** empower your agents to **interact effectively** with various **external applications**!

### Install Packages & Connect a Tool

<Tip>Goal: Allow agents to interact with GitHub projects, like starring a repository using natural language & Julep Agent</Tip>

These commands set up your environment to enable smooth interactions between Julep and GitHub.

<CodeGroup>
  ```bash Run Command 
  pip install composio-julep

  # Connect your GitHub so agents can interact with it
  composio-cli add github 

  # Check all supported apps 
  composio-cli show-apps
   ```
</CodeGroup>

<Steps>

<Step title="Import Base Packages & Create Default Julep Agent">
<CodeGroup>
```python Default Imports & Configuration
from julep import Client
from composio_julep import ComposioToolset, App
import os

api_key = os.getenv("JULEP_API_KEY")
base_url = os.getenv("JULEP_API_URL")
client = Client(api_key=api_key, base_url=base_url)

# Initialise the Composio Tool Set
composio_tools = ComposioToolset()
```
</CodeGroup>
</Step>
<Step title="Configure Julep Agent with Composio Tools">
<CodeGroup>
```python Agent Configuration
# Register the GitHub app with the right settings
composio_tools.register_tools(tools=[App.GITHUB])

# Create and configure the Julep agent
agent = client.agents.create(
    name="Jessica",
    about="Tech entrepreneur with a focus on sustainability and AI.",
    default_settings={
        "temperature": 0.7,
        "top_p": 1,
        "min_p": 0.01,
        "presence_penalty": 0,
        "frequency_penalty": 0,
        "length_penalty": 1.0,
        "max_tokens": 150
    },
    model="gpt-4",
    tools=composio_tools,
)
```
</CodeGroup>
</Step>
<Step title="Execute the Task via Julep Agent"> 
<CodeGroup>
```python Julep Agent Executes the Task

situation_prompt = "You are at a tech fair seeking innovative projects."
session = client.sessions.create(user_id="user-id", agent_id=agent.id, situation=situation_prompt)

user_msg = "Could you star the GitHub repository SamparkAI/composio_sdk?"

response = client.sessions.chat(
    session_id=session.id,
    messages=[
        {"role": "user", "content": user_msg, "name": "Sawradip"}
    ],
    recall=True,
    remember=True
)

print(response.messages)
```
</CodeGroup>
</Step>
<Step title="Check Response">
<CodeGroup>
```bash Check Agent Response
[{'content': 'I have starred the repository "composio_sdk" for you on GitHub under the account "SamparkAI".', 'role': 'agent'}]
```

</CodeGroup>
</Step>
</Steps>

### Use Specific Actions
<CodeGroup>
```bash Filter Specific Action
# To restrict agents from executing any actions, filter specific actions
composio_tools.register_tools(actions=[App.GITHUB_CREATE_ISSUE])
```
</CodeGroup>

### Use Specific Apps

<CodeGroup>
  ```bash Filter Specific App
  # To restrict agents from using all tools, filter specific tools
  composio_tools.register_tools(apps=[App.ASANA, App.GITHUB])
  ```
</CodeGroup>

### Filter apps actions by tags

<CodeGroup>
  ```bash Filter Actions by Tags
  composio_tools.register_tools(apps=[App.NOTIONBETA], tags=[Tag.NOTIONBETA_BLOCK])
  ```
</CodeGroup>
