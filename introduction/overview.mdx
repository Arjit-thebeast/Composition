## Guides
<CardGroup cols={3}>
<Card title="Integrations" icon="link" iconType="duotone" href="/guides/patterns/integrations">
   Integrations represent a tool configuration via which your user can authenticate and interact with the tool.
</Card>
<Card title="Actions" icon="screwdriver-wrench" iconType="duotone" href="/guides/patterns/actions">
   Actions are the operations that can be performed on a tool.
</Card>
<Card title="Triggers" icon="message-exclamation" iconType="duotone" href="/guides/patterns/triggers">
   Triggers are the events that can be used to invoke your agent.
</Card>
<Card title="Entities" icon="users" iconType="duotone" href="/guides/patterns/entity">
   An Entity in Composio represents a user or an organization. It is a collection of connected accounts that are managed by the entity.
</Card>
</CardGroup>
