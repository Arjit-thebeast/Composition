## Try Composio in under 5 mins.

Your one-stop platform for agent-tool interactions. Composio handles Authentication, Action Execution, Trigger Management, and Function Calling seamlessly.

<Tip>Goal: Perform actions on GitHub with Composio + Function Calling</Tip>

<Steps> 
<Step title="Setting Up GitHub Integration">
<AccordionGroup title="Select the option that suits you best.">
<Accordion title="Use Test Integration (Default) - Easy Start">
Please use `TestIntegration.GITHUB` for quick Setup 
</Accordion>
<Accordion title="Create your Integration - By Registering Github Developer App">
[Create Integration](https://docs.composio.dev/apps/github) _Copy your integration ID._
</Accordion>
</AccordionGroup>

</Step>

<Step title="Authenticating Github">
<CodeGroup>
```bash Install Composio SDK 
pip install composio-core
```
</CodeGroup>
Use `Github Integration` to Connect your `Github Account`

Modify *Composio* `api-key` 
<Tip> TestIntegration.*App* uses Composio built-in Integrations </Tip>
<CodeGroup>
```python Auth via OAuth 
from composio import Composio, TestIntegration, SchemaFormat

client = Composio("api-key") # init

#client.get_integration(****YourIntegrationID*****)
integration = client.get_integration(TestIntegration.GITHUB) # Modify

#Trying to initiate a new connection
connection_request = integration.initiate_connection(user_uuid = None)

print("Complete the auth flow, link: ", connection_request.redirectUrl)

```

</CodeGroup>
</Step>

<Step title="Using Function Call">
Modify `OPENAIKEY` before executing it.
<CodeGroup>
```python Task to Action Selection
from openai import OpenAI

my_task = "Star a repo SamparkAI/custom-auth-integrations on GitHub"

# Initialize the OpenAI client with your API key
openai_client = OpenAI(api_key="******OPENAIKEY******")

# Create a chat completion request to decide on the action
response = openai_client.chat.completions.create(
model="gpt-4-turbo-preview",
tools=actions, # Passing actions we fetched earlier.
messages=[
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": my_task}
  ]
)
```

</CodeGroup>
</Step>

<Step title="Handle Tool Calls - Execute Actions">
<CodeGroup>
```python Executing Selected Actions!
# Execute Function calls
execution_details = connected_account.handle_tools_calls(response)

print(execution_details)

print("Completed Task")

```

</CodeGroup>
</Step>
